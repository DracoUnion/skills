# ancestor-wisdom-inheritance

## When to Use

当需要在一个专业领域内进行创新、提升专业深度或建立行业影响力时，用于确保创新不脱离行业根本规律。description

## Core Logic

# 执行步骤

1.  **识别行业“祖师爷”与核心传统**
    *   **输入**：目标行业/领域名称。
    *   **操作**：研究并识别该领域的奠基人物（“祖师爷”）、关键历史发展阶段、以及被公认的经典理论、框架或作品（“传统”）。
    *   **判断标准**：能否清晰列出至少一位奠基人物和一项核心传统/原则。
    *   **输出**：一份关于该领域历史根基与核心原则的摘要。
    *   **常见误区**：将“祖师爷”狭义理解为神话人物，而忽略了真正的思想奠基者或范式开创者；只关注近期热点，忽视历史脉络。

2.  **解构传统中的“元模式”**
    *   **输入**：上一步获得的领域历史与核心传统摘要。
    *   **操作**：分析经典理论或作品，提炼出其底层结构、通用方法、核心原则或成功模式（即“元模式”）。例如，分析经典作品的故事框架、论证逻辑或表达技巧。
    *   **判断标准**：能否用简洁的语言描述出至少一个可迁移的“元模式”或核心原则。
    *   **输出**：一个或多个从传统中抽象出的“元模式”或核心原则列表。
    *   **常见误区**：停留在对传统内容的表面复述，未能进行抽象和模式提取；将特定时代背景下的具体做法误认为是永恒原则。

3.  **在“元模式”基础上进行时代化创新**
    *   **输入**：当前要解决的具体问题或创作目标，以及上一步提取的“元模式”/原则。
    *   **操作**：
        a. 将“元模式”或核心原则作为创新基底。
        b. 融入当前时代的新元素、新技术、新语境或新需求。
        c. 生成新的解决方案、内容或产品。
    *   **判断标准**：新产出物是否既能体现传统“元模式”的精髓，又具有鲜明的时代特征和创新点。
    *   **输出**：一个融合了传统根基与时代创新的方案或作品。
    *   **常见误区**：完全抛弃传统，进行无根由的“颠覆式”创新，导致产出物缺乏深度或偏离行业规律；或相反，僵化照搬传统，毫无新意。

4.  **通过“传道”巩固与传播**
    *   **输入**：基于传统创新的成果，以及对行业传统的理解。
    *   **操作**：主动向他人（如同行、后辈、受众）阐释你的成果如何继承并发展了行业传统。可以是通过创作（如作品、文章）、教学或公开分享等形式。
    *   **判断标准**：能否清晰地向他人说明你的工作与传统之间的承袭与创新关系。
    *   **输出**：一次或多次“传道”行为，以及由此带来的反馈或影响力。
    *   **常见误区**：只做不说，不主动建立自己与行业传统的联系；或传道时过于标榜自己，忽视对祖师爷和传统的尊重。

# 输出格式要求

当用户应用此模型后，应能输出：
1.  对所选领域“祖师爷”及核心传统的简述。
2.  从传统中提取出的“元模式”或核心原则。
3.  基于该“元模式”结合时代需求产生的具体创新方案或作品描述。
4.  （可选）计划或已实施的“传道”方式。
