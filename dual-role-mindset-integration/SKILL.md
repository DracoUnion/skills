# dual-role-mindset-integration

## When to Use

当感到内心焦虑、自我怀疑、内卷或动力枯竭时，用于调整心态，平衡内在接纳与外在专注description

## Core Logic

# 执行步骤

1.  **角色状态诊断与分离**
    *   **输入**：当前的负面情绪、压力情境或自我否定的具体想法。
    *   **操作**：
        a. **识别“内在父母/小孩”状态**：问自己“我现在是否在严厉地批评自己、感到不配得、过度担忧未来，或对自身某些特质感到困扰？”将困扰具体化（例如“我因为出身普通而感到自卑”），并记录这些自我对话。
        b. **识别“职业玩家”状态**：问自己“我现在是否因害怕失败而不敢行动、过度关注他人评价而分散精力、或对任务本身失去了兴趣和专注？”记录这些表现。
        c. **分离观察**：将这两种状态的影响书面分列，明确“哪些是自我接纳问题（内在父母/小孩），哪些是执行策略问题（职业玩家）”。
    *   **判断标准**：能够清晰区分出当前困扰中，源于自我接纳的部分和源于外部执行/专注的部分。
    *   **输出**：一份简单的诊断表，列明“内在父母/小孩”的困扰（一个明确的负面判断陈述）和“职业玩家”障碍。

2.  **追溯与接纳：重构“内在父母”叙事**
    *   **输入**：步骤1诊断出的“内在父母/小孩”的负面判断陈述。
    *   **操作**：
        a. **追溯源头**：反思这个判断最初可能来源于何处（如童年评价、社会标准、失败归因）。
        b. **承认与允许**：对内心的批评声音说：“我注意到我正在担心/批评自己……，我允许这种感受存在。”
        c. **重构叙事**：
            *   **IF** 该困扰与过去的经历（如家庭背景、早期教育）相关：尝试改变对这段经历的解读。例如，将“出身贫寒”重构为“培养了坚韧和从零开始的能力”。
            *   **ELSE IF** 该困扰与个人特点（如爱好广泛、性格内向）相关：尝试寻找该特质在特定情境下的优势。例如，将“爱好广泛”重构为“具有广泛的知识连接能力和探索精神”。
        d. **设定接纳锚点**：基于新叙事，总结一句自我接纳的肯定语，如“无论结果如何，我的价值是完整的”或“我已经尽力，可以放松”。想象一个支持性的“内在父母”会如何鼓励自己。
    *   **判断标准**：能够将当前自我判断与外部源头联系起来，并为原有“缺陷”找到一个积极或中性的重新定义；自我批判的强度减弱，能产生自我关怀的念头。
    *   **输出**：识别出的评价源头、一个新的积极/中性叙事，以及一句自我接纳的肯定语。

3.  **切换至“开放专注的职业玩家”**
    *   **输入**：步骤1诊断出的“职业玩家”障碍和步骤2输出的自我接纳肯定语。
    *   **操作**：
        a. **任务游戏化**：将当前任务重新定义为一场“游戏”或“挑战”，明确“游戏规则”（目标、规则）和想获得的“技能/装备”（要提升的能力）。
        b. **聚焦过程与实验**：将注意力从“必须成功”的结果，转移到“如何玩好下一步”的过程和“这次实验能学到什么”的好奇心上。
        c. **钝感力练习**：有意识地忽略或延迟处理与核心“游戏”无关的杂音（如无关评价、微小挫折），告诉自己“这与通关主线无关”。
    *   **判断标准**：对任务的感受从“负担”部分转变为“挑战”或“实验”，注意力更集中于行动本身。
    *   **输出**：一个重新定义后的“游戏化”任务描述及下一步具体行动。

# 输出格式要求
提供一份《心态切换与自我接纳记录》，包含：
1.  **情境与诊断**：简述引发心态波动的事件，并分列“内在父母/小孩”的困扰（负面判断）和“职业玩家”的障碍。
2.  **叙事重构与安抚**：
    *   这种看法可能来源于【识别出的源头】。
    *   现在，我选择将其重新理解为【新的积极/中性叙事】。
    *   内在父母安抚语：【记录步骤2中产生的自我接纳肯定语】。
3.  **职业玩家任务重构**：记录步骤3中“游戏化”后的任务描述。
4.  **即刻行动**：基于重构后任务，列出接下来1小时内可执行的一个小动作。

[CRITIC] 步骤2中，当困扰既不属于“过去经历”也不属于“个人特点”时（例如，对未来的抽象焦虑），重构叙事的IF-ELSE分支逻辑未能完全覆盖，可能导致用户在此步骤操作不明确。建议补充一个默认的“ELSE”分支，例如“ELSE：尝试从更长远或更广阔的视角看待当前困扰，寻找其可能带来的成长或启示”。
