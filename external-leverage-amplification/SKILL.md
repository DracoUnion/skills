# external-leverage-amplification

## When to Use

当个人或组织在特定领域已具备基础能力，但遭遇瓶颈或需要实现非线性突破时，调用此模型以识别并整合外部高能资源来放大自身能力。description

## Core Logic

# 执行步骤

1.  **识别核心瓶颈与目标差距**
    *   **输入**：当前个人/组织的核心能力评估、阶段性目标、与目标之间的显性差距（如技术短板、信息滞后、资源不足），或对观察对象（成功案例或自身现状）的初步了解。
    *   **操作**：明确界定在现有路径下难以克服的关键障碍或期望实现的非线性增长点。同时，建立“增长加速度”分析框架，理解其依赖于三个条件的正反馈循环：**A. 更高站位**（在系统中的有利位置）、**B. 更多实战机会**（高质量、高频率的实践）、**C. 更优方法**（通过实践迭代出的有效策略）。
    *   **判断标准**：能够清晰、具体地描述“如果仅依靠内部努力，将面临何种具体困难或缓慢进展”。能够清晰区分“站位、机会、方法”三个条件，并理解其循环关系。
    *   **常见误区**：将结果误认为原因，或只关注单一条件（如只强调努力/方法）。
    *   **输出**：一份明确的“瓶颈/差距定义书”及初步的三要素分析框架。

2.  **扫描与诊断外部环境与内部循环**
    *   **输入**：上一步输出的“瓶颈/差距定义书”及三要素框架。
    *   **操作**：
        *   **外部扫描**：系统性扫描所处生态系统（行业、跨领域、历史、全球范围），寻找可能包含解决方案或赋能要素的“高能节点”（如前沿技术、先进方法论、关键信息渠道、稀缺资源、卓越个体或组织）。
        *   **内部诊断**：针对“站位、机会、方法”三个条件，逐一评估满足程度，分析三者是否形成有效正反馈循环，并识别增长的主要瓶颈或循环断裂点。
        *   **阻力识别**：若目标推进受阻，暂停直接行动，分析各方显性诉求及其成立前提，定位阻碍目标达成的**最关键核心阻力**。
    *   **判断标准**：能够列出至少三个与“瓶颈/差距”潜在相关的外部高能节点。能够对三要素做出“是/否/部分”的判断并找到证据。识别出的核心阻力能解释“为何直接方法无效”。
    *   **输出**：一份“外部高能节点扫描清单”、一份“三要素与循环状态诊断报告”、一份“核心阻力分析清单”。

3.  **评估、筛选与策略制定**
    *   **输入**：“外部高能节点扫描清单”、“三要素与循环状态诊断报告”、“核心阻力分析清单”。
    *   **操作**：
        *   **外力评估**：对外部高能节点进行评估（匹配度、可行性、杠杆效应），筛选出1-2个最具潜力的“目标外力”。
            *   **IF** 节点整合难度极高但潜在回报巨大 → 考虑分阶段整合或寻找替代节点。
            *   **ELSE** (节点匹配且可行) → 将其确定为“目标外力”。
        *   **循环策略**：基于三要素诊断，明确策略方向。
            *   **IF** 三要素强循环 → 结论：处于加速区，应巩固循环。
            *   **ELSE IF** 某要素明显薄弱 → 结论：该要素是主要瓶颈，需优先突破。
            *   **ELSE IF** 循环断裂 → 结论：需修复循环链路。
        *   **竞争策略**：若涉及长期竞争，识别核心绩效指标(KPI)，对比自身与对手的长期平均值与波动性。
            *   **IF** 自身均值略高且波动性显著更低 → 采用“稳定性优先”策略。
            *   **ELSE IF** 自身波动性远高于对手 → 转向“消除波动，追求稳定输出”。
            *   **ELSE** (全面落后) → 此路径不适用，需解决基础问题。
        *   **阻力化解策略**：针对核心阻力，设计“潜沟通”干预方案（通过肢体语言、语气语调或间接行为释放信号），而非直接语言说服。
    *   **判断标准**：筛选出的“目标外力”能明确解释将如何解决瓶颈。形成了明确的增长循环优化策略、竞争优势构建策略或阻力化解方案。
    *   **输出**：明确的“目标外力”定义、增长优化策略、边际优势竞争策略或“潜沟通”干预方案。

4.  **设计整合路径与杠杆点**
    *   **输入**：步骤3输出的各项策略与“目标外力”。
    *   **操作**：规划将外部力量与自身能力结合，或执行既定策略的具体路径。识别关键的“杠杆点”——即投入较小努力即可引发较大正向变化的环节（如接入信息网络、采用标准化技术、建立关键合作关系、修复核心循环断点、实施稳定性训练、执行一次关键的“潜沟通”）。
    *   **判断标准**：整合或执行路径包含具体的行动步骤，并明确指出其中的“杠杆点”及其预期作用机制。
    *   **输出**：一份包含关键杠杆点的“整合与行动计划”。

5.  **执行、反馈与迭代**
    *   **输入**：“整合与行动计划”。
    *   **操作**：执行计划，并建立监测机制，观察实际影响。
    *   **IF** 放大效应显著/循环增强/优势巩固/阻力化解 → 巩固成果，探索优化。
    *   **ELSE IF** 效应不明显或出现偏差 → 返回步骤3，重新评估策略选择或调整路径。
    *   **ELSE** (情况复杂僵持) → 考虑“带着问题生活”，暂时搁置或转换战场，避免陷入“问题洁癖”。
    *   **判断标准**：能够通过关键指标或质性反馈，判断行动是否开始产生预期的非线性增长、瓶颈突破或阻力化解迹象。
    *   **输出**：效果评估报告及下一步决策（深化、调整、放弃或转换）。

# 输出格式要求

向用户呈现一个清晰的“能力放大与问题破解方案”，根据侧重点选择包含以下部分：
1.  **瓶颈与诊断**：简述识别的核心瓶颈、三要素循环状态或核心阻力。
2.  **核心策略**：明确推荐整合的外部力量、增长循环优化方向、竞争策略或“潜沟通”干预方案。
3.  **关键杠杆点**：指出行动计划中最关键的1-2个杠杆点。
4.  **预期跃迁**：描述成功实施后，能力、增长或局面可能实现的非线性变化。

[CRITIC] 合并后的Skill试图涵盖四个不同但相关的模型（外力整合、增长循环分析、边际优势、阻力化解），导致触发条件、前提和步骤变得复杂且宽泛，用户在实际调用时可能难以确定具体适用场景和入口。步骤逻辑虽试图串联，但分支过多，可能影响可操作性。建议拆分为更聚焦的独立Skill，或明确此为“战略问题解决元框架”，需用户根据初始问题选择进入不同分析分支。
