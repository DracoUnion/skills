# foundational-skill-isolation

## When to Use

当在复杂技能体系中表现不稳定，或过度依赖“状态/手感”时，用于识别和巩固核心能力description

## Core Logic

# 执行步骤

1.  **解构表现，识别“原始能力”**
    *   **输入**：一项复杂的整体表现或技能（如“投篮命中率”、“演讲感染力”）。
    *   **操作**：分析该表现由哪些最基础、最原子化的动作或能力要素构成。追问：“剥离所有环境、状态、运气因素后，最不可或缺的那个基础动作是什么？”
    *   **输出**：一个或多个被识别出的“原始能力”或“核心技能包”条目。
    *   **判断标准**：识别出的能力是否足够基础、纯粹，且是该领域高手公认的基石（如篮球的“篮下投篮”、演讲的“清晰吐字”）。

2.  **隔离训练，对抗“手感欺骗”**
    *   **输入**：上一步识别出的“原始能力”。
    *   **操作**：设计专门的练习，仅针对这项原始能力进行重复训练。练习环境应尽量简单，排除干扰，目的是追求该基础动作的绝对稳定性和极高的成功率，而非炫技或复杂应用。
    *   **输出**：针对一项原始能力的、聚焦的练习方案。
    *   **判断标准**：练习方案是否剥离了非必要的复杂情境，直指基础动作的机械重复与精度提升。

3.  **建立“基础锚点”意识**
    *   **输入**：在复杂情境或高压状态下进行表现。
    *   **操作**：当表现出现波动、手感不佳或信心受挫时，主动在思维中调用已被隔离训练过的“原始能力”要点。将其作为调整状态的“锚点”，通过回忆和执行最基础的要领来找回稳定感。
    *   **输出**：一次基于基础要领的临场调整。
    *   **判断标准**：能否在表现失常时，有意识地从追求“好状态”转向执行“基础动作”。

# 输出格式要求

向用户提供一个“核心能力加固”计划，包含：
1.  **需加固的复杂技能**：用户当前关注的不稳定技能领域。
2.  **被识别的原始能力**：列出1-3项从该技能中分解出的最基础、最核心的能力要素。
3.  **隔离训练建议**：为每项原始能力设计一个简单、可重复的针对性练习。
4.  **心理提示语**：提供一句在状态波动时可自我提醒的话，用于关联到基础锚点（例如：“回到篮下一米”）。
