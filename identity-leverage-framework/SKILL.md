# identity-leverage-framework

## When to Use

当需要解决特定人生或职业挑战（如快速学习、处理复杂关系、管理情绪、提升工作乐趣）时，通过切换或引入新身份视角来获得杠杆效应。description

## Core Logic

# 执行步骤

1.  **识别核心挑战**
    *   **输入**：当前面临的具体问题或情境描述。
    *   **操作**：明确界定需要解决的问题是什么（例如：“如何快速掌握新技能？”、“如何处理与同事的冲突？”、“如何应对工作压力？”）。
    *   **判断标准**：能够用一句话清晰陈述挑战。
    *   **输出**：一个清晰定义的问题陈述。

2.  **匹配赋能身份**
    *   **输入**：上一步定义的问题陈述。
    *   **操作**：从以下四种身份中选择一个或多个，作为审视和解决该问题的“透镜”：
        *   **艺术家**：适用于需要创造力、独特性、与时代趋势共舞或建立个人风格的挑战。
        *   **外交家**：适用于涉及复杂人际关系、沟通、谈判或需要建立广泛连接的挑战。
        *   **内在父母**：适用于管理自我情绪、接纳不完美、进行自我激励或保持内心稳定的挑战。
        *   **玩家**：适用于将任务游戏化、寻找乐趣、设定挑战目标或保持探索热情的挑战。
    *   **判断标准**：所选身份能对问题提供新的、建设性的思考角度。
    *   **输出**：一个或多个选定的“赋能身份”。

3.  **应用身份视角重构问题**
    *   **输入**：问题陈述 + 选定的赋能身份。
    *   **操作**：以选定身份的典型思维方式和行为模式，重新框架（Reframe）原始问题。
        *   **IF** 身份是“艺术家”：问题重构为“如何用独特/创造性的方式表达或解决这件事？”
        *   **IF** 身份是“外交家”：问题重构为“如何平衡各方利益，达成共赢或有效沟通？”
        *   **IF** 身份是“内在父母”：问题重构为“如何无条件地接纳现状，并鼓励自己采取积极行动？”
        *   **IF** 身份是“玩家”：问题重构为“如何将这个过程设计成一场有趣、有奖励的游戏？”
    *   **判断标准**：重构后的问题视角明显不同于初始的个人化/情绪化视角。
    *   **输出**：一个基于新身份视角重构后的问题。

4.  **生成身份化行动方案**
    *   **输入**：重构后的问题。
    *   **操作**：基于新身份的逻辑，构思具体的行动步骤。思考：“如果我是[选定身份]，我会怎么做？”
    *   **判断标准**：行动方案符合该身份的典型行为特征，且直接针对重构后的问题。
    *   **输出**：一套初步的、基于身份视角的行动策略或计划。

# 输出格式要求
以清晰的结构化方式呈现分析结果：
1.  **识别出的挑战**：[复述步骤1的问题陈述]
2.  **建议的赋能身份**：[列出步骤2选定的身份]
3.  **身份视角下的问题重构**：[展示步骤3重构后的问题]
4.  **初步行动思路**：[简述步骤4生成的行动方案要点]
