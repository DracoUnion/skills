# interest-first-communication

## When to Use

需要与陌生人（尤其是可能影响你命运的关键人物）进行高效沟通时description

## Core Logic

# 执行步骤

1.  **识别沟通对象的核心关切**
    *   **输入**：沟通对象的身份、背景、公开言论、已知信息，以及社交环境（如线上社群、线下活动、工作场景）。
    *   **操作**：分析输入信息，推断对方当前最关心、最在意或最可能产生兴趣的话题领域（如工作目标、个人利益、行业趋势、潜在收益等）。同时，观察并识别潜在的、有价值的陌生人连接对象。
    *   **判断标准**：能够明确陈述出1-2个对方可能关心的具体话题方向，并明确至少一个希望建立连接的目标对象及场景。
    *   **输出**：一个或多个“对方关切点”假设，以及目标连接对象及场景。

2.  **准备沟通切入点与素材**
    *   **输入**：步骤1输出的“对方关切点”假设、目标场景，以及你希望沟通的自身核心议题或信息。
    *   **操作**：
        *   **关联议题**：寻找自身议题与对方关切点之间的连接点。思考你的议题如何能影响、服务于或关联到对方的关切。
        *   **准备素材**：准备与目标对象可能共同关心的话题、行业动态、热点事件或中性趣闻，作为开启或维持对话的素材。避免依赖仅对熟人有意义的个人化故事。
    *   **判断标准**：
        *   能够用一句话清晰说明“我的[自身议题]如何与你的[对方关切点]相关”。
        *   准备的素材是否具有普适性、时效性或对目标群体有价值。
    *   **输出**：一个初步的“关联陈述”（沟通切入点），以及一份非个人化的聊天素材清单。

3.  **构建并执行沟通开场**
    *   **输入**：步骤2输出的“关联陈述”和聊天素材清单。
    *   **操作**：
        *   **IF** 沟通场景允许（如书面沟通或非即时对话），可设计一个基于“关联陈述”的简洁开场白，并模拟对方视角审视其吸引力和相关性。
        *   **ELSE**（即时对话或群体社交场景），在脑海中快速演练开场白，确保其自然、简洁且直指对方关切。随后，在目标场景中使用准备的素材发起一个新话题，或就现有话题贡献有价值的观点。
    *   **判断标准**：开场白能在一两句话内，从对方关切切入，并自然过渡到自身议题；在群体场景中，对话是否被激活（如有人回复、点赞、进一步提问）。
    *   **输出**：一个经过初步验证的沟通开场策略，或一个已激活的对话线程。
    *   **分支处理**：
        *   **IF**（在群体场景中）成功引发讨论或获得积极反馈 **THEN** 进入步骤4。
        *   **IF**（在群体场景中）未引发反响 **THEN** 返回步骤2，调整或更新聊天素材，或考虑更换连接场景。

4.  **进行价值交换式对话**
    *   **输入**：已激活的对话线程，或一对一沟通建立后的对话机会。
    *   **操作**：在对话中专注于信息、观点或资源的交换。提问以了解对方，分享知识以提供价值，避免将对话引向冗长且私密的“彻夜长谈”模式。
    *   **判断标准**：对话是否在有限时间内完成了有意义的信息交换，且双方感到有所收获。
    *   **输出**：一次完成的价值交换，以及可能的后续联系点（如关注、好友申请、约定下次交流主题）。

# 输出格式要求

根据沟通场景，选择以下一种格式输出：

**A. 针对高效沟通/关键人物场景：**
以结构化要点或简洁段落形式呈现：
1.  识别出的**对方潜在关切点**。
2.  建议的**沟通切入点**（即关联自身议题与对方关切的陈述）。
3.  **示例开场白**（可选，但建议提供）。

**B. 针对建立连接/扩大影响力场景：**
以结构化摘要形式输出本次连接行动的结果，包括：
1.  连接对象/场景。
2.  使用的核心话题。
3.  互动反馈摘要。
4.  是否建立了可持续的弱关系连接点。

[CRITIC] 步骤3的分支处理逻辑仅针对“群体社交场景”的反馈进行了定义，对于“一对一即时沟通”或“书面沟通”场景下，若开场未达到预期效果（如对方无回应、反应冷淡），缺少明确的回退或调整路径。建议补充通用性的失败处理机制，例如：“若初步沟通未达预期，可返回步骤1，重新评估关切点，或返回步骤2，准备新的关联角度与素材。”
