# personal-pe-valuation

## When to Use

当需要评估个人或项目的长期价值增长潜力时description

## Core Logic

# 执行步骤

1.  **计算静态市盈率 (PE)**
    *   **输入**：个人/项目的当前“总市值”（如：市场估值、年薪总额、项目总估值）和最近一年的“净利润”（如：净收入、项目净收益）。
    *   **操作**：使用公式 `PE = 总市值 / 净利润` 进行计算。
    *   **判断标准**：得出一个具体的数值（如 10, 50, 100）。
    *   **输出**：静态市盈率 (PE) 数值。
    *   **常见误区**：使用不准确的“总市值”或“净利润”数据；将短期波动收入误认为稳定净利润。

2.  **评估增长潜力 (G)**
    *   **输入**：个人/项目过去 3-5 年的“净利润”历史数据，或基于可靠依据的未来 3-5 年“净利润”增长率预测。
    *   **操作**：计算或预测“净利润”的年化复合增长率 (G)。
    *   **判断标准**：得出一个百分比数值（如 15%, 29%, 50%）。
    *   **输出**：年化净利润增长率 (G) 数值。
    *   **常见误区**：仅用单一年份的爆发式增长来代表长期增长率；对未来增长做出过于乐观且无依据的预测。

3.  **计算市盈率相对盈利增长比率 (PEG)**
    *   **输入**：步骤1输出的 `PE` 值和步骤2输出的 `G` 值。
    *   **操作**：使用公式 `PEG = PE / G` 进行计算。**注意**：G 为百分比，代入计算时需转换为小数（如 15% 应代入 0.15）。
    *   **判断标准**：得出一个具体的数值（如 0.5, 1.0, 2.0）。
    *   **输出**：PEG 比率数值。
    *   **常见误区**：忽略增长率单位（百分比），错误地将 G=15% 代入为 15 进行计算。

4.  **确定基准增长率并评估相对态势**
    *   **输入**：所处宏观环境（如国家）或所在行业的长期平均增长率数据（例如GDP增速、行业平均增速）。
    *   **操作**：
        *   将宏观/行业平均增长率设定为基准线。
        *   将步骤2输出的自身增长率 (G) 与基准增长率进行比较，计算相对倍数或差值。
    *   **判断标准**：是否获得了可靠、长期的基准增长数据，并准确计算了相对关系。
    *   **输出**：基准增长率数值，以及当前增长率与基准的相对关系（例如，是基准的1.5倍）。

5.  **进行综合价值判断与战略制定**
    *   **输入**：步骤3输出的 `PEG` 值，以及步骤4输出的“当前增长率与基准的相对关系”。
    *   **操作**：
        *   **A. 基于PEG的价值评估**：
            *   **IF** `PEG < 1`：表明当前估值相对于增长潜力可能被低估，具有较高投资价值。
            *   **ELSE IF** `PEG ≈ 1`：表明估值与增长基本匹配，处于合理区间。
            *   **ELSE IF** `PEG > 1`：表明当前估值可能已透支未来增长，存在高估风险。
        *   **B. 基于相对增长态势的战略选择**：
            *   **IF** 当前增长率显著高于基准（例如达到基准的1.5-2倍或更高）：
                *   采取“**持续优势**”策略。核心是保持当前赛道的专注，优化内部效率，追求稳定、可持续的增长，而非盲目追求更高增速或切换赛道。
            *   **ELSE IF** 当前增长率接近或低于基准：
                *   采取“**开放探索**”策略。核心是积极寻找新赛道、新定位或新模式，通过结构性调整寻求增长机会。
    *   **判断标准**：综合PEG估值和增长相对水平，得出价值结论和战略方向。
    *   **输出**：价值评估结论（低估/合理/高估）与核心战略方向（持续优势/开放探索）。

6.  **设定合理增长目标与行动建议**
    *   **输入**：步骤5输出的战略方向。
    *   **操作**：
        *   **IF** 战略方向为“**持续优势**”：
            *   **目标设定**：将增长目标设定在略高于行业/领域平均水平的可持续区间（例如基准增速的1-2倍），接受增速从高点逐步回归但仍保持相对优势。
            *   **行动建议**：聚焦于提升运营效率、巩固核心竞争力、深化现有市场。
        *   **ELSE IF** 战略方向为“**开放探索**”：
            *   **目标设定**：将增长目标设定为在新赛道达到或超过该赛道的平均增长水平，重点关注增长趋势的扭转和市场份额的获取。
            *   **行动建议**：投入资源进行新领域探索、模式创新、建立新的能力或合作伙伴关系。
        *   **通用建议**：若PEG显示“高估”，需警惕增长不及预期的风险，并审视支撑高估值的故事是否可持续；若PEG显示“低估”，在采取“开放探索”策略时可能意味着更高的潜在回报。
    *   **判断标准**：设定的目标与行动是否与所选战略方向及价值评估结论一致。
    *   **输出**：符合长期可持续性原则的具体增长目标范围，以及1-2条关键行动建议。

# 输出格式要求

向用户呈现一个结构化的综合评估与战略报告，包含：
1.  **核心指标**：清晰列出计算出的 PE、G、PEG 以及选定的基准增长率。
2.  **综合诊断**：
    *   **价值评估**：基于PEG，给出“低估”、“合理”或“高估”的判断。
    *   **态势评估**：基于相对增长率，指出处于“优势区间”或“待提升/探索区间”。
3.  **战略建议**：明确建议采取“持续优势策略”或“开放探索策略”。
4.  **目标与行动**：
    *   提出未来阶段的可持续增长目标区间。
    *   给出1-2条与战略相匹配的具体后续行动建议。
5.  **重要提示**：提醒用户该分析模型的局限性（如依赖数据准确性、忽略非财务与定性因素、均值回归的长期性等）。

[CRITIC]
1.  **逻辑闭环缺陷**：步骤5中，基于PEG的价值评估结论（低估/合理/高估）与基于相对增长态势的战略选择（持续优势/开放探索）是两条独立的决策路径，在步骤6中才进行简单关联。未提供明确的决策矩阵或优先级逻辑来处理两者可能产生的矛盾情景（例如，PEG显示“高估”但增长态势处于“优势区间”时应如何权衡）。
2.  **缺失分支处理**：步骤5的B部分（战略选择）只定义了“显著高于基准”和“接近或低于基准”两种情况，未明确“略高于基准”（如1.2倍）这类灰色地带的处理逻辑。
3.  **缺少错误处理**：整个流程未包含数据校验或输入错误的处理方案（例如，净利润为负或零时如何计算PE；增长率数据不足或不可靠时的备选方案）。
