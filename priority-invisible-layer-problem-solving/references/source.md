# 推荐序二　涌动的暗物质里，藏着生命演化的秘密

易仁永澄　永澄文化创始人，个人成长研究者

一口气读完书，我冒出一句：“如果我早做到这些就好了！”

想要写些什么，但又感觉有无数个声音在呐喊，好像有无数个小人在内心叫着：“把这一章再看一下，把这个观点再读一下，把这个故事摘抄一下。”

于是，又用了两周时间，把书认真地读了几遍，同事问我：“你最近怎么了？好像换了个人，思考方式都变了。”

仅仅两周时间，变化就被他人看出来了。

我想要揭开改变的秘密，了解究竟是何种暗物质在牵引着我？

第一，这本书提出了一个生命演化的问题。

如何开挂成为高手，并成为可复制的科学，成为时代的宠儿？

我很喜欢这个问题，好的问题引领生命的方向。

多年来我一直在研究一件事：“如何活出值得拥有的人生”，并形成了一个框架：

·内在和外在：内在稳定，外在富足；

·当下和未来：当下不杂，未来不迎；

·坚持和享受：坚持必败，享受必成。

在三个维度建立是非观、时空观和系统观，有一个共同的方案：“优先解决不可见层的问题”。

时代是一个多维度复杂不可见的东西，我们必然需要站在不可见层迎接。

第二，尊重并且跨越人的核心恐惧。

一个好的问题，通常都不会立刻揭示，往往要践行数十年才有答案。

不是大家意愿不足，而是好问题引发了“我不能、我不行、我不配”的恐惧，从而放弃。

本书最大的特征是：全局观。

无论是哪个问题，书中都会给你一个全局观，从小白视角到高手视角完整剖析。

“学习究竟为了什么？武学高手都有哪些层次？这个时代都有哪些改变？如何做出更加正确的选择？”这些问题中都透露着体谅和包容。

每一份恐惧都被拥抱，每一份弱小都被抚慰，每一份期待都被支持，每一种悲欢离合都被同情，才有力量继续前行。

第三，拉阔和填充读者的内在空间。

我有一种既熟悉又陌生的阅读体验，熟悉得温柔，陌生得清爽。

温柔和清爽皆来自一扇六边形的网：

·如何识局破局？

·如何获取想象力？

·如何恰当取舍？

·如何适应竞争？

·如何过滤信息？

·如何化解阻力？

网里的每一个真问题，必然会带来全新的答案。

商业、管理、投资的水，流入个体成长的河流，或静水流深，或汹涌澎湃。

你会忘了自己是在看书。

第四，提供差异化的方案。

问题提出了，一定要有解决路径。

标准答案很容易给，但笑泪与共、游刃有余的思维路径，更实用。

·给平凡人提供故事和案例带动其启发。

·给践行者提供方法和策略帮助其应用。

·给思考者提供问题和视角帮助其思考。

·给领导者提供原则和理念帮助其引领。

道术器用，多元而有序、严谨而欢脱，这就是本书的独特价值。
