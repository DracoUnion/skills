# quantity-over-initial-quality

## When to Use

当在技能学习或创意创作初期，过度追求单次产出质量导致进展缓慢时description

## Core Logic

# 执行步骤

1.  **评估当前阶段与目标**
    *   **输入**：当前从事的技能学习或创作任务。
    *   **操作**：明确当前是处于“探索与积累期”（需要拓宽视野、积累经验）还是“精修与交付期”（需要打磨单一作品质量）。
    *   **判断标准**：如果目标是提升整体能力、寻找方向或积累经验，而非立即产出唯一终极作品，则进入步骤2。
    *   **输出**：确认为“探索与积累期”。

2.  **设定数量导向的周期目标**
    *   **输入**：步骤1的确认。
    *   **操作**：为一个特定周期（如一周、一个月）设定明确的产出数量目标（如“完成30个草图”、“拍摄100张照片”、“撰写20个短文开头”）。质量要求仅为“完成”。
    *   **判断标准**：目标是否清晰量化，且重点完全放在“完成的数量”而非“单品的完美度”上？
    *   **输出**：一个明确的、以数量为核心的行动目标。

3.  **执行高频、快速的实践循环**
    *   **输入**：步骤2设定的数量目标。
    *   **操作**：以最快速度执行任务，完成单个产出。禁止在单个作品上反复修改、纠结。完成一个，立即开始下一个。尝试不同的方法、角度或风格。
    *   **判断标准**：是否保持了较高的产出频率，并抑制了在过程中追求完美的倾向？
    *   **输出**：一系列已完成但未必完美的初级作品或实践记录。

4.  **进行批量回顾与模式识别**
    *   **输入**：步骤3产出的一批作品。
    *   **操作**：在完成一定数量（如10个、20个）后，集中回顾所有产出。不评判单个好坏，而是寻找重复出现的模式、共同的问题、意外的亮点或个人的偏好。
    *   **判断标准**：是否从批量作品中发现了仅凭思考无法获得的、关于技能或创作的“真实感受”和规律？
    *   **输出**：基于大量实践总结出的经验模式、待改进的共性问题和潜在的优势方向。

5.  **将识别出的模式应用于下一轮实践**
    *   **输入**：步骤4识别出的模式、问题和方向。
    *   **操作**：将上轮总结的经验，有意识地带入下一轮“数量导向”的实践中，进行微调和新尝试。循环步骤3和步骤4。
    *   **判断标准**：新的实践循环是否融入了上一轮的真实反馈，而非空想？
    *   **输出**：进入更高效的、基于真实反馈的迭代学习循环。

# 输出格式要求
输出应包含：
1.  **周期与数量目标**：明确的时间周期和计划完成的数量。
2.  **已完成数量**：当前周期内实际完成的作品/实践数量。
3.  **关键模式发现**：从批量回顾中总结出的1-3个最重要的模式、问题或感受。
4.  **下一轮调整计划**：基于上述发现，计划在下一轮实践中如何调整尝试方向。
