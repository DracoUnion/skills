# self-evaluation-via-reversed-criteria

## When to Use

当需要自我反思或提升个人在合作、社交中的长期价值时description

## Core Logic

# 执行步骤

1.  **内化外部评估标准**
    *   **输入**：“厚道且灵活”评估模型中对A、B、C、D四类人的定义与评价。
    *   **操作**：深刻理解每类行为模式背后的品质（如A类的知识准确但可能刻板，B类的不诚实，C类的诚实但被动，D类的诚实且主动）。
    *   **判断标准**：能够清晰复述并解释四类行为的核心区别与价值高低。
    *   **输出**：内化的行为-品质对照表。

2.  **情境映射与自我审视**
    *   **输入**：当前或过往的某个具体互动情境（如回答他人提问、处理未知任务、团队协作中的承诺）。
    *   **操作**：回顾自己在该情境中的言行，对照步骤1的内化标准，判断自己的行为最接近哪一类（A/B/C/D）。
        *   **IF** 发现自己常表现为B类（胡诌、掩饰） → **THEN** 识别出“不厚道”的风险点。
        *   **IF** 发现自己常表现为C类（诚实但止步） → **THEN** 识别出“灵活性不足”的改进点。
        *   **IF** 发现自己常表现为A类（仅复述已知） → **THEN** 思考如何增加“灵活”的维度。
        *   **IF** 发现自己常表现为D类（诚实且主动求解） → **THEN** 确认并强化该行为模式。
    *   **判断标准**：能够客观地将自己的具体行为归入某一类别，并识别出对应的优势或短板。
    *   **输出**：对自身在特定情境下“厚道”与“灵活”表现的自评结果及关键发现。

3.  **制定行为改进策略**
    *   **输入**：步骤2得出的自评结果与关键发现。
    *   **操作**：针对待改进的类别（B、C、A），设计具体、可操作的行为替代方案，目标是向D类行为靠拢。
        *   对于“不厚道”（B类）：练习说“我不知道”，并停止编造。
        *   对于“不灵活”（C类）：在承认不知道后，强制增加一个动作，如“我可以去查一下”或“我请教一下再回复您”。
        *   对于“仅复述”（A类）：在给出标准答案后，尝试补充个人理解、应用场景或提出一个相关的新问题。
    *   **判断标准**：为每个待改进点制定了至少一个明确、可执行的新行为指令。
    *   **输出**：一份针对性的个人行为调整清单。

# 输出格式要求
以反思报告的形式呈现：
- **内化的评估标准**：[简述A/B/C/D四类的核心特征]
- **自我审视情境**：[描述所回顾的具体事件]
- **行为归类与自评**：
  - 我当时的行为属于：[A/B/C/D]类
  - 厚道度自评：[高/中/低] 理由：[...]
  - 灵活度自评：[高/中/低] 理由：[...]
- **改进策略**：
  - 识别出的主要短板：[如“倾向于C类行为”]
  - 具体改进行动：[列出1-2条可立即实施的新行为]
