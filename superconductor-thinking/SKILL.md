# superconductor-thinking

## When to Use

当面对他人或外部环境的优势/成功时，产生比较心理或防御心态的场景description

## Core Logic

# 执行步骤

1.  **识别比较信号**
    *   **输入**：接收到关于他人/外部环境优势的信息（如他人成就、城市发展、他人观点）。
    *   **操作**：觉察自身第一反应。判断是否出现以下任一信号：
        *   **IF** 反应是“寻找对方/事物的缺点或自身优点进行反驳/平衡”。
        *   **IF** 反应是“感到不适、嫉妒或防御，并试图合理化自身现状”。
        *   **ELSE** 反应是“感到欣赏、好奇或受到启发”。
    *   **判断标准**：能清晰识别出自身情绪和思维倾向属于“比较/防御”还是“开放/学习”。
    *   **输出**：明确的信号分类（比较防御型 / 开放学习型）。

2.  **中断自洽循环**
    *   **输入**：步骤1的输出为“比较防御型”信号。
    *   **操作**：主动停止正在进行的“错位比较”或“自我合理化”的内心对话。明确意识到这种行为会隔绝信息与成长机会。
    *   **判断标准**：能够有意识地暂停防御性思维，为转换思维模式创造空间。
    *   **输出**：中断旧的思维循环，进入待转换状态。

3.  **切换至超导思维**
    *   **输入**：步骤2的“待转换状态”或步骤1的直接输出为“开放学习型”信号。
    *   **操作**：执行思维转换。
        *   **IF** 面对他人/环境的优势：承认对方/事物的优点（“你好我认”），并聚焦于“我可以从中学到什么？”（“虚心学习模仿”）。
        *   **IF** 自身处于优势位置时被他人学习：乐于分享自身经验与方法，不藏私（“乐于分享”）。
    *   **判断标准**：思维焦点从“比较/捍卫自我”成功转移到“识别价值点”和“建立学习或分享连接”上。
    *   **输出**：一个具体的学习问题（如“他成功的核心方法是什么？”）或一个分享意愿。

4.  **执行价值连接**
    *   **输入**：步骤3输出的“具体学习问题”或“分享意愿”。
    *   **操作**：
        *   **IF** 输入是学习问题：采取行动获取答案，如深入研究、请教、模仿实践。
        *   **IF** 输入是分享意愿：选择合适的渠道与方式，真诚地进行经验或信息分享。
    *   **判断标准**：将思维转化为具体行动，并产生实际的信息输入或输出。
    *   **输出**：获得新知识/技能，或完成一次有效分享。

# 输出格式要求
用户应能陈述：
1.  触发的具体情境（遇到了谁的什么优势）。
2.  识别的初始反应类型（比较防御型 / 开放学习型）。
3.  最终采取的超导思维行动（学习了什么 / 分享了什么）。
4.  （可选）行动带来的具体收获或反馈。
