# Write Code Analysis Skill

将源代码转换为深入浅出的技术教程，采用《Claude Code完全指南》的技术写作风格。

## 核心特点

1. **源码驱动**：从实际代码出发，逐行解析
2. **生活类比**：用日常场景建立技术直觉
3. **结构统一**：每章遵循固定的教学模板
4. **渐进难度**：从基础概念到深入实现
5. **架构可视化**：丰富的图表和流程图

## 使用方法

在对话中触发以下关键词：
- "根据源码写教程"
- "源码分析教程"
- "代码解析书籍"
- "技术书籍写作"
- "深入浅出源码"

## 写作方法论

### 章节结构模板
```
标题 —— 形象副标题
├── 学习目标（3-5点）
├── 生活类比（贯穿全文）
├── 源码解析
│   ├── 模块地图
│   ├── 类型定义（先画图纸）
│   ├── 逐行解析
│   └── 完整数据流
├── 设计中的取舍
├── 动手练习
└── 本课小结
```

### 必须标注的内容
- 源码文件路径：`src/xxx/yyy.ts`
- 核心类/接口：\`ClassName\`
- 关键函数：\`functionName()\`
- 重要常量：\`CONSTANT_NAME\`

### 生活类比示例
| 技术概念 | 生活类比 |
|---------|---------|
| Agent Loop | 厨师做菜 |
| BashTool安全检查 | 银行柜台验证 |
| QueryEngine | 超级管家 |
| 权限系统 | 门卡等级 |
